## Run locally

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file and add your Perplexity API key:
```
PERPLEXITY_API_KEY=your_key_here
```

3. Start the server:
```bash
npm start
```

4. Open `http://localhost:3000` in your browser.
